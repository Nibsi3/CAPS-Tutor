# **App Name**: CAPS Tutor

## Core Features:

- Grade/Subject Selection: Allow students to select their grade and subject from the CAPS curriculum.
- Interactive Practice: Enable students to answer questions and receive immediate feedback (correct/incorrect) with step-by-step explanations using a tool.
- Lesson Hub: Provide a searchable lesson content with embedded practice questions and quizzes.
- Adaptive Exam Generator: Generate custom past papers focused on the student's weak topics and provides timed practice mode.
- Progress Dashboard: Track lessons completed, mastery per learning objective, time spent, and historical performance for each student.
- Syllabus Uploader: Allow teachers to upload CAPS PDFs or structured content for indexing and usage by the AI using RAG.
- Student Management: Enable teachers to view class progress, problematic topics, and export reports. This will use a Cloud SQL database.

## Style Guidelines:

- Primary color: Deep teal (#008080) to reflect the educational and slightly technical aspect of the app, offering a sense of reliability and calm.
- Background color: Light teal (#E0F8F8), creating a soft and clean backdrop that reduces visual strain and supports prolonged learning sessions.
- Accent color: Mustard yellow (#FFDB58) to highlight interactive elements and important feedback, providing clear visual cues for students.
- Body text: 'PT Sans', sans-serif. Headline font: 'Space Grotesk', sans-serif. The former provides clarity, the latter has a technical feel.
- Code snippets: 'Source Code Pro' to clearly present code, separate from learning materials.
- Consistent, clear icons that match the CAPS syllabus themes for easy navigation.
- Subtle transitions and animations to provide feedback, like a slide-in panel with results, enhancing engagement.
